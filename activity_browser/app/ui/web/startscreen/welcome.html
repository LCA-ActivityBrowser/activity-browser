Welcome to the Activity Browser !

The Activity Browser is an open source graphical user interface for the Brightway2 advanced life cycle assessment framework.

 

Brightway is great for advanced LCA modeling. The Activity browser provides a graphical interface for many standard LCA tasks and thus should contribute to your productivity with brightway2.

Features

Management of brightway2 projects, databases and activities

Fast calculation of LCA results and contribution analyses

Monte Carlo simulation

Sankey diagrams and a Graph explorer

Development

The Activity Browser is an open source project written in Python and being developed by a number of researchers in their free time mainly from ETH Zurich and University of Leiden (CML). Development is ongoing and new features are constantly being added. 

Contributing

As the Activity Browser is a true open source project, contributions from the LCA community are very welcomed. There are several ways in which you can contribute:

  • by raising issues related to bugs or desirable features directly on our GitHub site (no programming skills required)
  • by making a contribution to the code

Please see our page on contributing for more detailed instructions.
